## 🆘 Quick Help

**Can't find what you're looking for?**

1. Check the table of contents in each doc (they all have one)
2. Use Ctrl+F / Cmd+F to search within files
3. Check [TROUBLESHOOTING_NATIVE.md](TROUBLESHOOTING_NATIVE.md) first for errors
4. Read [WHAT_HAPPENS_WHEN.md](WHAT_HAPPENS_WHEN.md) to understand the system

**Still stuck?**

- Run `npx cap doctor` to check your setup
- Check console logs in Xcode/Android Studio
- Google the exact error + "capacitor"
- Visit [Capacitor Docs](https://capacitorjs.com/docs)
